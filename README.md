# XhsCommentHelper

## 功能

- 悬浮窗筛人助手
- 读取当前评论
- 读取当前屏幕已加载的评论和回复
- 打开用户主页
- 读取主页公开显示的性别/IP 线索
- 人工标记合适/不合适
- 生成私信草稿
- 复制、打开私信入口或填入私信框
- 人工确认发送

## 权限

- 辅助功能权限：用于读取当前屏幕控件文本，并尝试填入私信草稿。
- 悬浮窗权限：用于在小红书 App 上方显示筛人助手工具条。

## 使用步骤

1. 打开 App，设置固定话术。
2. 开启辅助功能权限。
3. 开启悬浮窗权限。
4. 点击开启悬浮窗。
5. 打开小红书 App 评论区。
6. 点击读取评论/回复列表。
7. 点击下一位主页。
8. 点击读取主页信息，查看公开显示的性别/IP 线索。
9. 人工判断是否合适。
10. 点击生成私信。
11. 点击打开私信入口。
12. 点击填入私信框。
13. 人工确认发送。

## 注意

本工具不自动发送私信，不批量处理用户，不自动判断性别/IP，仅作为用户手动操作的提效辅助工具。

当前版本只读取无障碍树里当前已加载的评论和回复，不会自动无限滚动抓取全帖。

如果自动读取评论失败，可以先在小红书中手动复制评论文本，再点击悬浮窗的读取评论/回复列表按钮。工具会从剪贴板兜底读取。

如果自动填入私信失败，工具会保留私信内容到剪贴板，请手动粘贴并确认后再发送。
